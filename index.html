Projects

Hash Function

The purpose of this hash functions project is to explore how hash functions work and why they are important. A hash function takes some input, like a word, and turns it into a short string of numbers and letters.

Matching Card

This project was made to create fun and simple games where players flip over cards to find matching pairs. This project helps us learn how to use logic, memory, and design in a game. By building this project, we get better at programming and understand how to create interactive applications.

Password Generator

This password generator was the first project for us as an introduction on how to program in JavaScript. This project is important for making strong, random passwords to help keep accounts safe. This project helped us learn about random number generation, user input, and basic security.
